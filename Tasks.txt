    - Include a DB connectivity check (EF Core ping)

[ ] Standardise DTO mapping strategy
    - DashboardService manually constructs DTOs (80+ lines); other services use AutoMapper
    - Pick one approach and apply consistently across all services
    - If keeping AutoMapper, move manual mappings into MappingProfile
    - If dropping AutoMapper, remove the dependency and map explicitly everywhere

[ ] Angular HTTP error interceptor
    - Add a single HttpInterceptor that catches 4xx/5xx responses
    - Show a snackbar or toast with a user-friendly message

[ ] Split large components into sub-components
    - sprint-detail.component.ts is 671 lines with 5 unrelated responsibilities
    - Extract: MemberCardComponent, WorkloadSummaryComponent, SprintRetroComponent
    - Each sub-component takes inputs via @Input signals and emits events upward
    - Reduces cognitive load and makes individual sections independently testable

[ ] Replace inline onmouseenter/onmouseleave with CSS hover
    - Templates use onmouseenter="this.style.background='...'" throughout
    - Replace with :hover rules in the component styles array
    - Removes plain JS string handlers from Angular templates

[ ] Decouple from shared Postgres network
    - API currently joins external postgres_default network owned by another service
    - Consider running a dedicated postgres instance in this compose file
    - Or document the dependency explicitly and add a healthcheck on the DB container

[ ] Separate migration step from app startup
    - db.Database.Migrate() on startup is a race condition risk if ever scaled to 2+ instances
